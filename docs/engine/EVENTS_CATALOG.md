# Каталог событий EventBus

Шина событий (`EventBus`) — основной механизм обмена сообщениями между системами без создания жестких зависимостей.

## Природный плагин (com.island.nature.event)

### AnimalBornEvent
Публикуется при рождении нового животного.
- **Поля:**
  - `animal: Organism` — объект новорожденного животного.
- **Использование:** Статистика, логгер, визуализация спецэффектов.

### AnimalDiedEvent
Публикуется при гибели животного.
- **Поля:**
  - `animal: Organism` — объект погибшего животного.
  - `cause: DeathCause` — причина смерти (голод, возраст, съеден и т.д.).
- **Использование:** Статистика, очистка ресурсов, триггеры для других систем.

## Городской плагин (com.island.simcity.event)

### ResidentBornEvent
Публикуется при появлении нового жителя в городе.
- **Поля:**
  - `resident: SimEntity` — объект жителя.
- **Использование:** Демографическая статистика.

### ResidentDiedEvent
Публикуется при смерти жителя.
- **Поля:**
  - `resident: SimEntity` — объект жителя.
- **Использование:** Обновление численности населения, статистика смертности.

## Как подписаться (Java)
```java
eventBus.subscribe(AnimalDiedEvent.class, event -> {
    System.out.println("Animal died because of " + event.getCause());
});
```

## Как опубликовать (Java)
```java
eventBus.publish(new AnimalBornEvent(newAnimal));
```

## Рекомендации по расширению
При добавлении нового события:
1. Создайте класс в пакете `com.island.{plugin}.event`.
2. Событие должно быть неизменяемым (immutable, желательно `record` или финальные поля).
3. Добавьте описание в этот каталог.
